Const after

  • after(ms: number, objOrFn?: any, label?: any): AwaitableObservable
    • Defined in src/utils.ts:38

    Returns an Observable of the value, or result of the function call, after the number of milliseconds given. After is lazy and cancelable! So nothing happens until .subscribe is called explicitly (via subscribe) or implicitly (toPromise(), await). For a delay of 0, the function is executed synchronously when .subscribe is called.

    Parameters

    • ms: number
    • Optional objOrFn: any
    • Optional label: any

    Returns AwaitableObservable

    An Observable of the object or thunk return value. It is 'thenable', so may also be awaited directly.

    // Examples:
// awaited Promise
await after(100, () => new Date())
// unawaited Promise
after(100, () => new Date()).toPromise()
// unresolving Promise
after(Infinity, () => new Date()).toPromise()
